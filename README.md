## Installation
```
git clone https://github.com/waleed-aldhahi/LSLS.git
cd LSLS
pip install -r requirements.txt  # pip install -e .
python run.py
```
## Or
```
pip install git+https://github.com/waleed-aldhahi/LSLS.git
```
## Run 
```
python run.py
```
